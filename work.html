Simplified Bash group project c bash

Launches executables and builtins. Handles pipes, logical operators and subshell. Expands env var and *. Echo, cd, pwd, export, unset, env, exit are implemented. Made with Arthur Prahin .

Wireframe Renderer c computer graphics macOS

Renders a simple wireframe model of a landscape from given 3D coordinates. User can translate, zoom, rotate the rendered model and choose its projection type.

Mylibc c

A few functions of the C standard library recoded by me. Features my implementation of printf.

Overwatch Landing Page javascript bootstrap html css

A tribute to a popular multiplayer FPS "Overwatch" that uses Blizzard assets. Showcases css animations and parallax.

Weather Web App javascript bootstrap html css api
